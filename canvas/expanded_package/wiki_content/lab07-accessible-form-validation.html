Lab 07 – Accessible Form Validation

Purpose

This lab gives students a constrained practice space for the week's concepts. The emphasis is on seeing, changing, and explaining what the browser does.

Tasks

  1. Create or update your repo branch for this lab.
  2. Build the required interface or script in small increments.
  3. Test keyboard access, responsiveness, and error states.
  4. Write a brief notes section describing what worked, what broke, and how you fixed it.
  5. Commit meaningful progress with short messages.

Deliverable

Submit a deployed link, repository link, and a short process reflection.

Suggested rubric

  • Functionality
  • Code organization
  • Accessibility and UX clarity
  • Reflection quality
