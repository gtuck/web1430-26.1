# WEB 1430 Project Memory

## Project Type
Complete college course build for WEB 1430 – Client Side Web Development at Weber State University.
4-credit, fully online, async Canvas course. One git commit: `cdc0913 Initial WEB1430 course build`.

## Repository Structure
- `textbook/chapters/` — 12 Markdown chapter files (ch01–ch12)
- `labs/` — 14 lab files (lab00–lab13)
- `assignments/` — 6 assignment files
- `projects/` — 3 project files (project-1, project-2, final-project)
- `course/` — syllabus.md, schedule.md, learning_outcomes.md, import_to_canvas.md
- `canvas/` — Pre-built Canvas export package (imscc + expanded XML/HTML)
- `reports/` — ID review report (claude-id-report.md)

## Critical Known Issues (from ID report, March 2026)
1. **All 12 textbook chapters are template stubs** — identical boilerplate (key ideas, mental model, working habits, common mistakes, vocabulary, reflection, checklist). Only title and opening sentence differ.
2. **All 14 labs are identical templates** — same 5 generic tasks, same deliverable, same rubric across all labs. No specific instructions per lab.
3. **All 6 assignments share the same requirements/rubric** — only summary paragraph differs.
4. **Rubrics are category lists only** — no performance levels or point values.
5. **Assignment 1 has a sequencing error** — requires JavaScript but is due before JS is introduced (Week 3).
6. **No starter code or example files** exist anywhere in the repo.
