# Project 1 – Interactive Style Guide

## Summary
Design and build a style guide that demonstrates typography, color, spacing, buttons, cards, navigation, and at least three scripted interface patterns.

## What to build
- A working client-side experience
- Clean project structure
- Meaningful commit history
- Deployment link
- Short rationale describing audience, decisions, and known limitations

## Requirements
- Semantic HTML
- Responsive behavior
- Accessible interaction patterns
- Readable JavaScript with named functions or modules
- Testing notes and final review checklist

## Submission
Submit the live URL, repo URL, and a concise reflection.

## Rubric highlights
- Technical completion
- UX clarity
- Maintainability
- Accessibility
- Reflection and communication

## Milestones
- Proposal
- Build Checkpoint
- Final Submission
